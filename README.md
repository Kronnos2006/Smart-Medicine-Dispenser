# Documentación del Proyecto Smart Medicine Dispenser

## Introducción
El Dispensador de Medicamentos Inteligente es un dispositivo diseñado para ayudar a los pacientes a gestionar su medicación de manera eficaz. Proporciona recordatorios, seguimiento y dispensación automática de medicamentos.

## Características Principales
- **Dispensación Automática**: Administra la cantidad correcta de medicación en el momento adecuado.
- **Recordatorios de Medicación**: Envía notificaciones al usuario cuando es el momento de tomar su medicación.
- **Seguimiento del Uso**: Registra el uso de medicamentos y envía informes a los cuidadores o médicos.
- **Interfaz de Usuario Amigable**: Sencilla y fácil de usar, adecuada para todas las edades.

## Instalación
1. Conectar el dispensador a la corriente eléctrica.
2. Configurar la conexión a Internet.
3. Descargar la aplicación asociada desde la tienda correspondiente.
4. Seguir las instrucciones en la pantalla para configurar su cuenta y añadir medicamentos.

## Uso
Para utilizar el Dispensador de Medicamentos Inteligente:
- Asegúrese de tener la medicación correcta en el dispensador.
- Ajuste los horarios de dispensación a través de la aplicación.
- Reciba notificaciones para cada dosis programada.
